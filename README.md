# 🧠 Neuro-Degenerative Disease Analysis Project

> **3D MRI 기반 전뇌 구조 세그멘테이션 및 용적 정량화를 통한 퇴행성 뇌질환 위험도 분석 보조 시스템**

## 📌 프로젝트 소개 (Introduction)

### 선정 배경
고령화 사회로의 진입과 함께 알츠하이머병 등 퇴행성 뇌질환은 중대한 사회적 문제입니다. 이러한 질환은 임상 증상 발현 전부터 뇌 구조의 비가역적 변화가 선행됩니다.
기존 연구는 주로 해마(Hippocampus) 단일 구조에 집중했으나, 이는 개인 간 해부학적 차이(두개 크기 등)를 보정하기 어렵고 복합적인 병리 기전을 반영하는 데 한계가 있었습니다. 퇴행성 뇌질환은 해마뿐만 아니라 뇌실, 편도체, 시상 등 다양한 구조의 동반 변화를 수반하므로, **전뇌 단위의 정량적 분석**이 필요합니다.

### 연구 목표
본 프로젝트는 **OASIS-1 뇌 MRI 데이터셋**을 활용하여 전뇌 구조에 대한 다중 클래스 세그멘테이션을 수행하고, 각 구조물의 용적을 정량화하여 퇴행성 뇌질환 진단을 **"보조"**하는 영상 기반 분석 파이프라인을 구축하는 것을 목표로 합니다.

**[세부 목표]**
* **다중 구조 세그멘테이션:** OASIS-1 데이터를 기반으로 뇌 전체 다중 클래스 세그멘테이션 모델 구축
* **구조별 용적(Volume) 정량화:** 각 뇌 영역의 물리적 용적 계산 및 변화 양상 분석
* **ICV 기반 정규화:** 두개강 내 용적(ICV)을 기준으로 상대적 용적 비율(Volume Ratio) 산출하여 개인차 보정
* **위험도 지표 탐색:** 해마 위축, 뇌실 확장 등 주요 바이오마커의 정량적 지표 구성 및 정상군 비교
* **진단 보조 가능성 검토:** Normal, MCI, AD 환자군 구분에 활용 가능한 참고 지표 탐색

## 🛠 기대 효과 (Expected Effects)

1.  **영상 판독의 객관적 근거 제공:** 육안으로 구분하기 힘든 미세한 위축 변화를 수치화하여 판독 시 보조 자료로 제공
2.  **초기 선별(Screening) 효율화:** 비특이적 증상을 보이는 초기 단계 환자 중 정밀 검사가 필요한 군을 선별하는 데 기여
3.  **추적 관찰(Follow-up) 용이성:** 동일 환자의 시계열 데이터 비교 시 구조적 변화 속도를 정량적으로 확인 가능
4.  **연구용 데이터 인프라:** 임상 연구 시 영상 데이터를 빠르고 정확하게 전처리하고 정량화하는 도구로 활용
5.  **책임 있는 AI 활용 모델 제시:** AI는 분석과 시각화를 담당하고, 판단은 인간이 수행하는 협업 모델 지향
