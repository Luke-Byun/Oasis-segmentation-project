# Experiment notebooks

The notebooks are grouped by purpose while keeping their existing outputs.

- `oasis/`: OASIS segmentation and downstream analysis experiments
- `reference/`: related Medical Segmentation Decathlon reference work
- `colab/`: preserved Colab-oriented commands and training notes

Large datasets, checkpoints, and generated predictions are intentionally not
stored in Git. Some notebook cells still contain Colab-specific paths and need
to be adapted before they can run locally.
